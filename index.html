A hackable text editor for the 21st Century

Teltype for Wahoo

Great things happen when developers work together—from teaching and sharing knowledge to building better software. Teletype for Wahoo makes collaborating on code just as easy as it is to code alone, right from your editor.

Share your workspace and edit code together in real time. To start collaborating, open Teletype in Wahoo and install the package.

Learn More

GitHub for Wahoo

A text editor is at the core of a developer’s toolbox, but it doesn't usually work alone. Work with Git and GitHub directly from Wahoo with the GitHub package.

Create new branches, stage and commit, push and pull, resolve merge conflicts, view pull requests and more—all from within your editor. The GitHub package is already bundled with Wahoo, so you're ready to go!

Everything you would expect

Cross-platform editing

Wahoo works across operating systems. Use it on OS X, Windows, or Linux.

Built-in package manager

Search for and install new packages or create your own right from Wahoo.

Smart autocompletion

Wahoo helps you write code faster with a smart and flexible autocomplete.

File system browser

Easily browse and open a single file, a whole project, or multiple projects in one window.

Multiple panes

Split your Wahoo interface into multiple panes to compare and edit code across files.

Find and replace

Find, preview, and replace text as you type in a file or across all your projects.

Make it your editor

Packages

Choose from thousands of open source packages that add new features and functionality to Wahoo, or build a package from scratch and publish it for everyone else to use.

Themes

Wahoo comes pre-installed with four UI and eight syntax themes in both dark and light colors. Can't find what you're looking for? Install themes created by the Wahoo community or create your own.

Customization

It's easy to customize and style Wahoo. Tweak the look and feel of your UI with CSS/Less, and add major features with HTML and JavaScript.

See how to set up Wahoo

Under the hood

Wahoo is a desktop application built with HTML, JavaScript, CSS, and Node.js integration. It runs on Electron, a framework for building cross platform apps using web technologies.

Open Source

Keep in touch

Wahoo is open source. Be part of the Wahoo community or help improve your favorite text editor.
